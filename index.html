Senior Admin

  • debug - Usage: /<command>
    For developers only - debug things via reflection.
  • doom - Usage: /<command> <playername>
    For the bad admins
  • wipeuserdata - Usage: /<command>
    Removes essentials playerdata
  • fuckoff - Usage: /<command> <on [radius (default=25)] | off>
    You'll never even see it coming.
  • deafen - Usage: /<command>
    Make some noise.
  • rawsay - Usage: /<command> <message>
    Broadcasts the given message. Supports colors.
  • wipeflatlands - Usage: /<command>
    Wipe the flatlands map. Requires manual restart after command is used.
  • lockup - Usage: /<command> <all | purge | <<partialname> on | off>>
    Block target's minecraft input. This is evil, and I never should have wrote it.
  • kicknoob - Usage: /<command>
    Kick all non-superadmins on server.

SuperAdmin

TotalFreedom Super Admin Commands


  • kick - Usage: /<command> <player> [reason] - Aliases: k
  • Kick a player.
  • adminchat - Usage: /<command> [message...] - Aliases: o, ac :
  • AdminChat - Talk privately with other admins. Using <command> itself will toggle AdminChat on and off for all messages.
  • rollback - Usage: /<command> <[partialname] | undo [partialname] purge [partialname] | purgeall> - Aliases: rb
  • Issues a rollback on a player
  • glist - Usage: /<command> <purge | ban <username> [reason] | unban <username>>
  • Bans or unbans any player, even those who are not logged in anymore.
  • tban - Usage: /<command> <player> [reason] - Aliases: noob
  • Temporarily bans a player for five minutes.
  • setlimit - Usage: /<command> - Aliases: setl, swl
  • Sets everyone's Worldedit block modification limit to the default.
  • setspawnworld - Usage: /<command>
  • Set world spawnpoint.
  • protectarea - Usage: /<command> <list | clear | remove <label> | add <label> <radius>>
  • Protect areas so that only superadmins can directly modify blocks in those areas. WorldEdit and other such plugins might bypass this.
  • jumppads - Usage: /<command> <on | off | info | sideways <on | off> | strength <strength (1-10)>> - Aliases: launchpads, jp
  • Manage jumppads
  • mobpurge - Usage: /<command> - Aliases: mp
  • Purge all mobs in all worlds.
  • premium - Usage: /<command> <player> - Aliases: prem
  • Validates if a given account is premium.
  • stfu - Usage: /<command> [[-s] <player> [reason] | list | purge | all] - Aliases: mute
  • Mutes a player with brute force.
  • toggle - Usage: /<command> [option] [value] [value]
  • Toggles TotalFreedomMod settings
  • gtfo - Usage: /<command> <partialname>
  • Makes someone GTFO (deop and ip ban by username).
  • opall - Usage: /<command> [-c | -s]
  • Op everyone on the server, optionally change everyone's gamemode at the same time.
  • smite - Usage: /<command> <player> [reason]
  • Someone being a little bitch? Smite them down...
  • blockcmd - Usage: /<command> <-a | purge | <player>> - Aliases: blockcommands, blockcommand, bc, bcmd
  • Block all commands for a specific player.
  • warn - Usage: /<command> <player> <reason>
  • Warns a player.
  • consolesay - Usage: /<command> <message...> - Aliases: csay
  • Telnet command - Send a chat message with chat formatting over telnet.
  • deop - Usage: /<command> <playername>
  • Deop a player.
  • gcmd - Usage: /<command> <fromname> <outcommand>
  • Send a command as someone else.
  • permban - Usage: /<command> reload
  • Manage permanently banned players and IPs.
  • lastcmd - Usage: /<command> <player>
  • Show the last command that someone used.
  • mobpurge - Usage: /<command> - Aliases: mp
  • Purge all mobs in all worlds.
  • trail - Usage: /<command> [off]
  • Pretty rainbow trails.
  • undisguiseall - Usage: /<command> - Aliases: uall
  • Undisguise all players on the server
  • moblimiter - Usage: /<command> <on|off|setmax <count>|dragon|giant|ghast|slime>
  • Control mob rezzing parameters.
  • expel - Usage: /<command> [radius] [strength]
  • Push people away from you.
  • cmdspy - Usage: /<command> - Aliases: commandspy
  • Spy on commands
  • health - Usage: /<command>
  • View ticks-per-second
  • logs - Usage: /<command> [off]
  • Register your connection with the TFM logviewer.
  • wildcard - Usage: /<command> [fluff] ? [fluff] ?
  • Run any command on all users, username placeholder = ?.
  • denick - Usage: /<command>
  • Essentials Interface Command - Remove the nickname of all players on the server.
  • orbit - Usage: /<command> <target> [<<power> | stop>]
  • POW!!! Right in the kisser! One of these days Alice, straight to the Moon!
  • nickclean - Usage: /<command> - Aliases: nc
  • Essentials Interface Command - Remove distracting things from nicknames of all players on server.
  • whohas - Usage: /<command> <item> clear - Aliases: wh
  • See who has a block and optionally clears the item.
  • opme - Usage: /<command>
  • Automatically ops user.
  • announce - Usage: /<command> <message>
  • Make an announcement
  • purgeall - Usage: /<command>
  • Superadmin command - Purge everything! (except for bans).
  • stop - Usage: /<command>
  • Kicks everyone and stops the server.
  • adminmode - Usage: /<command> [on | off]
  • Close server to non-superadmins.
  • qdeop - Usage: /<command> <partialname>
  • Quick De-Op - deop someone based on a partial name.
  • findip - Usage: /<command> <player>
  • Shows all IPs registered to a player
  • freeze - Usage: /<command> [target | purge] - Aliases: fr
  • Freeze players (toggles on and off).
  • invis - Usage: /<command> (smite)
  • Shows (optionally smites) invisisible players
  • gadmin - Usage: /<command> [list | [<kick | nameban | ipban | ban | op | deop | ci | fr | smite> <targethash>] ]
  • Use admin commands on someone by hash. Use mode 'list' to get a player's hash. Other modes are kick, nameban, ipban, ban, op, deop, ci, fr, smite.
  • setlimit - Usage: /<command> - Aliases: setl, swl
  • Sets everyone's Worldedit block modification limit to the default.
  • cage - Usage: /<command> <purge | off | <partialname> [outermaterial] [innermaterial]>
  • Place a cage around someone..
  • entitywipe - Usage: /<command> <carts> - Aliases: ew, rd
  • Remove various server entities that may cause lag, such as dropped items, minecarts, and boats.
